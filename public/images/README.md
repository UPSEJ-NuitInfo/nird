# Images & Assets

Ce dossier contient les ressources visuelles du projet.

## Structure

```
images/
├── hero/           # Images page d'accueil
├── badges/         # Badges de l'Académie
├── levels/         # Illustrations des niveaux
└── testimonials/   # Photos des témoignages
```

## Sources d'images libres

### Illustrations
- **unDraw** : https://undraw.co/ (personnalisables, SVG)
- **Storyset** : https://storyset.com/ (illustrations animées)
- **Humaaans** : https://www.humaaans.com/ (personnages)

### Photos
- **Unsplash** : https://unsplash.com/ (haute qualité)
- **Pexels** : https://www.pexels.com/ (variées)
- **Pixabay** : https://pixabay.com/ (nombreuses options)

### Icônes
- **Font Awesome** (déjà intégré via CDN)
- **Heroicons** : https://heroicons.com/ (SVG)
- **Feather Icons** : https://feathericons.com/ (minimalistes)

### Badges & Achievements
- Créer avec **Canva** (gratuit) : https://www.canva.com/
- Templates de badges disponibles

## Dimensions recommandées

- **Hero** : 1920x1080px (bannières)
- **Badges** : 256x256px (PNG transparent)
- **Levels** : 800x600px (illustrations)
- **Testimonials** : 400x400px (portraits carrés)
- **Icons** : 64x64px ou SVG

## Optimisation

Avant d'ajouter des images :

1. **Compresser** avec TinyPNG : https://tinypng.com/
2. **Convertir en WebP** si possible (meilleure compression)
3. Garder les fichiers < 200KB chacun
